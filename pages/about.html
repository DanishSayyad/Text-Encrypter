About This Project

A secure, client-side encryption and decryption tool

What is Secret Encrypter?

Secret Encrypter is a web-based application that allows you to securely encrypt and decrypt text using custom encryption keys. All operations happen entirely in your browser, ensuring complete privacy and security.

Key Features

  • ✓
    Client-Side Processing

    All encryption and decryption happens in your browser. No data is sent to any server.

  • ✓
    Key-Based Encryption

    Use your own custom keys to encrypt and decrypt messages securely.

  • ✓
    Password Protection

    Access to the application is protected by SHA-256 hashed passwords stored only in your browser's local storage.

  • ✓
    No Registration Required

    Start using the app immediately without creating an account.

How to Use

  1. Set up your application password on first visit
  2. Choose to encrypt or decrypt text
  3. Enter your encryption key (remember it for decryption)
  4. Input your text and click the operation button
  5. Copy the result and use it as needed

Privacy & Security

Your privacy is paramount. This application:

  • Does not collect any personal information
  • Does not transmit data to external servers
  • Stores only hashed passwords in browser local storage
  • Runs entirely offline after initial page load

Important: We do not store any content you provide on our servers. Everything is stored only in your browser's local storage.
